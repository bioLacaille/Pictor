# Pictor-Biological-Analysis-Platform

Pictor-Biological-Analysis-Platform（无码生信分析平台），生物大数据分析于一体平台,集成多个软件工具、标准化分析流程，并可自主定制分析流程，同时采用可视化、便捷化操作，大幅度提升数据分析效率。
